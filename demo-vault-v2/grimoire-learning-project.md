# Grimoire Feature Showcase

This is the shippable project inside the demo vault. It shows how Grimoire holds a real working system: Markdown notes, journals, dreams, links, properties, search, command flows, canvases, attachments, audio capture, console notes, local agents, privacy boundaries, experience profiles, and portability.

## Start here

Open [[grimoire-start-here]] first, then keep [[grimoire-feature-tour]] beside it as the feature checklist.

- [[grimoire-feature-tour]] for the full app surface map.
- [[grimoire-markdown-learning]] for Markdown basics.
- [[grimoire-properties-and-types]] for frontmatter, custom types, icons, colors, and properties.
- [[grimoire-search-and-commands]] for Quick Open, command palette, and find flows.
- [[grimoire-journal-demo-2026-05-31]] for dated journal capture.
- [[grimoire-dream-demo-2026-05-31]] for dreams.
- [[grimoire-calendar-time-loom]] for metadata-only calendar and Time Loom proof.
- [[grimoire-links-and-backlinks]] for wikilinks, backlinks, and graph thinking.
- [[grimoire-console-and-agents]] for command notes and local AI agents.
- [[grimoire-agent-council]] for source-safe Council lanes, review-gated synthesis, and the live/not-live boundary.
- [[grimoire-local-agent-map]] for local assistant lanes and boundaries.
- [[grimoire-canvas-and-attachments]] for sketches and media.
- [[grimoire-crystallize-example]] for turning raw capture into a durable note.
- [[grimoire-audio-transcription]] for audio import, recording, transcript, and clean-note flow.
- [[grimoire-privacy-and-memory]] for Locality Firewall, Memory, and safe agent context.
- [[grimoire-portability-and-sync]] for Git, export/import, and object-storage readiness.
- [[grimoire-settings-and-themes]] for experience profiles, color mode, typography, providers, and API key settings.

## Why it exists

A new user should be able to open this project and learn Grimoire without reading external docs. Every linked note is ordinary Markdown, so users can edit, copy, rename, link, and delete the examples safely.
