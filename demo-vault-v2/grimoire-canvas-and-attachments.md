# Canvas and Attachments

Grimoire can keep sketches and attachments near the note that explains them.

## Attachment example

The existing fixture note [[grimoire-qa-reference]] embeds an image from `attachments/grimoire-reference.png`.

```markdown
![Fixture reference image](attachments/grimoire-reference.png)
```

## Canvas idea

Use a canvas when the thought is spatial:

- Draw a flow before writing it.
- Sketch how a project connects to people and responsibilities.
- Keep the visual artifact linked to the Markdown explanation.

## Practice

- [ ] Add a small sketch to a note.
- [ ] Link the sketch note back to [[grimoire-learning-project]].
- [ ] Keep the attachment local unless you intentionally export it.
