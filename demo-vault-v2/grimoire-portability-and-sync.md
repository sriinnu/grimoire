# Portability and Sync

The demo vault is meant to prove that Grimoire does not trap the user.

## Surfaces to inspect

- Local Markdown files on disk.
- Git status, commits, and conflict handling.
- Import from Markdown-oriented tools.
- Export to portable bundles.
- Object-storage readiness checks for S3, Azure, and compatible providers.
- Static or capsule-style sharing without leaking private files.

## Practice

Open this note from the file system and then from Grimoire. It should still make sense in both places.
