# Start Here - Learn Grimoire

Welcome to the demo vault. This project is the shippable, editable tour of Grimoire.

Grimoire is not only a notes app. It is a local Markdown workspace for thinking, writing, remembering, linking, journaling, tracking dreams, sketching, and asking local agents to help.

## Try this first

1. Open [[grimoire-feature-tour]] and scan the full product checklist.
2. Open [[grimoire-markdown-learning]] and edit one heading.
3. Create a wikilink to this note by typing `[[grimoire-start-here]]`.
4. Open [[grimoire-properties-and-types]] and inspect the frontmatter.
5. Open [[grimoire-journal-demo-2026-05-31]] and notice the launch-day date.
6. Open [[grimoire-dream-demo-2026-05-31]] and compare it with the journal entry.
7. Open [[grimoire-calendar-time-loom]] to see how dated metadata becomes a private calendar graph.
8. Open [[grimoire-links-and-backlinks]] to see why links matter.
9. Open [[grimoire-console-and-agents]] to learn how command notes and agents fit into the same project.

## What to notice

- The vault is just Markdown files on disk.
- Frontmatter gives notes useful structure.
- Wikilinks connect notes without locking them into one folder.
- Journal and dream entries stay private by default.
- Agent notes describe work without pretending the agent ran anything.
- Feature notes stay editable, so the demo is not a frozen help page.

## Keep going

When you are done, open [[grimoire-learning-project]] again and treat it like a real project dashboard.
