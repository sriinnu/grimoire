# Properties and Types

Grimoire keeps structure in frontmatter so the vault stays portable.

## What to inspect

- `type: Note` controls the lane and icon family.
- `belongs_to` makes this part of [[grimoire-learning-project]].
- `related_to` builds a looser relationship to the feature tour.
- `status`, `owner`, `priority`, and `color` appear in cards and panels.

## Type notes

The demo vault includes editable type definitions:

- [[Journal]] for the Journal lane.
- [[Dream]] for the Dream lane.
- [[Console]] for command notes.
- [[Agent]] for local agent lanes.

Open the Properties panel on this note and change one field. The Markdown file remains readable on disk.
