# Settings and Themes

Settings are part of the product story, not a drawer of toggles.

## Try this

- Switch theme presets and confirm the demo notes stay readable.
- Change editor font and line height, then save.
- Open provider settings and confirm API keys are handled separately from notes.
- Check privacy, workflow, sync, and appearance settings.

## What should never happen

No theme should make this vault unreadable, and no provider setting should write secrets into Markdown.
