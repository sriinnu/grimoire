# Privacy and Memory

Grimoire can give agents context without handing them the whole vault.

## Locality Firewall

- Public project notes can travel when the user asks.
- Private journals, dreams, and memories are withheld unless explicitly allowed.
- Agent context should name what was withheld instead of pretending nothing exists.

## Memory ledger

Memory is useful only when it is inspectable. Link durable facts to source notes, then let the app summarize what is safe to use.
