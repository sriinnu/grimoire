# Search and Commands

This note is for testing navigation surfaces without needing special data.

## Try this

- Search for `feature tour` and open [[grimoire-feature-tour]].
- Use Quick Open to jump to [[grimoire-canvas-and-attachments]].
- Use the command palette to create a note, switch views, or open settings.
- Use document find to search this note for `command palette`.

## Why it matters

A demo vault should be discoverable by search alone. If a user forgets where the tour starts, searching `Grimoire Feature Tour` should bring them back.
